Welcome to okscraper's documentation!
=====================================

Okscraper is a python module which helps standardize and simplify the scraping process of websites and documents.

If you are using django you should also check out `okscraper-django <http://okscraper-django.readthedocs.org/>`_ which provides some django-specific features on top of okscraper.

Installation: `pip install okscraper`

The code is on `GitHub <https://github.com/hasadna/okscraper>`_.
